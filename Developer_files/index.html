Wstęp

Tworzenie stron internetowych to zabawna i satysfakcjonująca umiejętność. W tym przewodniku omówimy podstawy tworzenia stron za pomocą HTML, CSS i JS.

Podstawy HTML

HTML (HyperText Markup Language) to standardowy język znaczników do tworzenia stron internetowych.

  • Strukturyzuj swoją treść za pomocą elementów takich jak <div>, <p> i <h1>.
  • Używaj atrybutów, aby dostarczyć dodatkowe informacje o elementach, np. class i id.
<!DOCTYPE html>
<html>
  <head>
    <title>Moja pierwsza strona</title>
  </head>
  <body>
    <h1>Witaj!</h1>
    <p>To jest moja pierwsza strona internetowa.</p>
  </body>
</html>

Podstawy CSS

CSS (Cascading Style Sheets) służy do stylizowania treści HTML.

  • Używaj selektorów, aby kierować się na elementy, np. body, .class i #id.
  • Stosuj style, takie jak kolory, czcionki i układy.
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f9;
  color: #333;
}

h1 {
  color: #0056b3;
}

Podstawy JS

JS (JavaScript) służy do funkcjonowania elementów strony.

  • Używaj stworzonych komend (Nie wymyślaj ich!), np. alert, prompt i document.
  • Możesz używać selektorów; QuerySelector i QuerySelectorAll.
  • Na końcu wyrażenia zawieraj ;, np. document.QuerySelectorAll("h1").html(color, red);
